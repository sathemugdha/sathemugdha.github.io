# Mugdha Arun Sathe - Portfolio

This is a professional portfolio website for Mugdha Arun Sathe, a Computer Science graduate student at San Jose State University.

## Features

- Responsive design that works on all devices
- Clean, modern interface with light theme
- Interactive navigation with smooth scrolling
- Contact form with validation
- Sections for education, experience, projects, and skills
- Links to LinkedIn and GitHub profiles

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- Vanilla JavaScript (no frameworks)
- Font Awesome icons
- Google Fonts (Poppins and Roboto)

## How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content to match your information

## Customization

To customize this portfolio for your own use:

1. **Update Personal Information**: Replace all personal details in `index.html` with your own
2. **Add Your Photo**: Replace the placeholder profile image in the hero section with your own photo
3. **Update Content**: Modify the education, experience, projects, and skills sections
4. **Change Colors**: Adjust the CSS variables in `style.css` to change the color scheme
5. **Update Links**: Update social media links in the contact section

## Hosting on GitHub Pages

1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to Repository Settings > Pages
4. Select the main branch as the source
5. Your portfolio will be available at `https://yourusername.github.io/repositoryname`

## Browser Support
